## About

The Weather API is designed to provide accurate and real-time weather data to users. It supports various features such as current weather conditions, forecasts, and more. The API is built to be lightweight and easy to integrate into web and mobile applications.

## Features

- Current weather data for any location
- 7-day weather forecast
- Search functionality for cities
- Support for multiple units (metric and imperial)

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript

- **APIs**: OpenWeatherMap API
